\doxysubsection{Detailed Description}
Get an OAuth2 token from an OAuth2 provider.
\begin{DoxyItemize}
\item Install this script on your server so that it\textquotesingle{}s accessible as \mbox{[}https/http\mbox{]}\+://$<$yourdomain$>$/$<$folder$>$/get\+\_\+oauth\+\_\+token.php e.\+g.\+: \href{http://localhost/phpmailer/get_oauth_token.php}{\texttt{ http\+://localhost/phpmailer/get\+\_\+oauth\+\_\+token.\+php}}
\item Ensure dependencies are installed with \textquotesingle{}composer install\textquotesingle{}
\item Set up an app in your Google/\+Yahoo/\+Microsoft account
\item Set the script address as the app\textquotesingle{}s redirect URL If no refresh token is obtained when running this file, revoke access to your app and run the script again.
\end{DoxyItemize}
