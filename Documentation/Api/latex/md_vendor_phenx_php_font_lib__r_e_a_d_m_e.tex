This library can be used to\+:
\begin{DoxyItemize}
\item Read True\+Type, Open\+Type (with True\+Type glyphs), WOFF font files
\item Extract basic info (name, style, etc)
\item Extract advanced info (horizontal metrics, glyph names, glyph shapes, etc)
\item Make an Adobe Font Metrics (AFM) file from a font
\end{DoxyItemize}

You can find a demo GUI \href{http://pxd.me/php-font-lib/www/font_explorer.html}{\texttt{ here}}.
